# Bharat Warriors Cricket Club Website

[![Website](https://img.shields.io/badge/Website-bharatwarriors.org-blue)](https://bharatwarriors.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Official website for **Bharat Cricket Club By UGNT** - A premier cricket club based in the Dallas-Fort Worth area, fostering cricket excellence and community spirit.

## 🏏 About

Bharat Warriors is a dynamic cricket club that brings together cricket enthusiasts from the Dallas-Fort Worth community. We participate in various leagues, organize tournaments, and promote the spirit of cricket through competitive play and community engagement.

## ✨ Features

- **Responsive Design**: Fully responsive website that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Interactive Gallery**: Photo galleries showcasing team moments and achievements
- **Player Profiles**: Dedicated section for team warriors and player information
- **Match Fixtures**: Up-to-date schedule of upcoming matches and tournaments
- **Achievements**: Showcase of team accomplishments and milestones
- **Contact Form**: Easy way for supporters and potential players to reach out
- **Partner Section**: Display of sponsors and partners supporting the club

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic web server (optional, for local development)

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Aksharnixglobal/bharatwarriors.org.git
cd bharatwarriors.org
```

2. Open the website:
   - **Option 1**: Simply open `index.html` in your web browser
   - **Option 2**: Use a local web server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser

## 📝 Content Updates

### Adding New Content

- **Images**: Add to the `images/` directory and reference in HTML
- **Player Profiles**: Update `warriors.html` with new player information
- **Fixtures**: Modify `fixtures.html` to add upcoming matches
- **Gallery**: Add new photos to `gallery.html`

### Styling

All custom styles are maintained in `styles.css`. The website uses:
- Color scheme: Navy blue (#1e3a4a), Gold (#D4AF37, #ffd700), Red (#dc3545)
- Font: Montserrat (Google Fonts)
- Responsive breakpoints for mobile, tablet, and desktop
