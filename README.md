# Bharat Warriors Cricket Club Website

Official website for Bharat Cricket Club by UGNT.

## 🚀 Quick Start with Visual Studio Code

This project is configured for Visual Studio Code with all the necessary settings and extensions!

### Prerequisites
- [Visual Studio Code](https://code.visualstudio.com/) installed on your computer

### Setup Instructions

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/Aksharnixglobal/bharatwarriors.org.git
   cd bharatwarriors.org
   ```

2. **Open in VS Code**:
   ```bash
   code .
   ```
   Or open VS Code and select File → Open Folder → select the project folder

3. **Install Recommended Extensions**:
   - When you open the project, VS Code will show a notification to install recommended extensions
   - Click **"Install All"** or go to Extensions panel (Ctrl+Shift+X) and install the workspace recommendations

4. **Start Development**:
   - Open `index.html`
   - Right-click in the editor and select **"Open with Live Server"**
   - Your browser will open at `http://localhost:5500` with live reload!

## 🛠️ Development Workflow

### Method 1: Live Server (Recommended)
1. Open any HTML file
2. Click "Go Live" in the status bar (bottom right)
3. Browser opens at `http://localhost:5500`
4. Edit files and see changes instantly!

### Method 2: Debug Mode
1. Press `F5`
2. Select "Launch Chrome against localhost"
3. Breakpoints and debugging available

### Method 3: Open in Browser
1. Right-click any HTML file
2. Select "Open with Default Browser"

## 🐛 Troubleshooting

### Live Server Not Working?
- Ensure the extension is installed
- Check if port 5500 is available
- Try restarting VS Code

### Extensions Not Installing?
- Check internet connection
- Open Extensions panel (Ctrl+Shift+X)
- Search and install manually

### Files Not Formatting?
- Install Prettier extension
- Check `.prettierrc` exists
- Ensure format on save is enabled
